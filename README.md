# Correlating Abstract Syntax Tree (AST) Divergence with Hallucination Rates in LLMs

This research project investigates whether the structural stability of code generated by Large Language Models (LLMs) correlates with their tendency to hallucinate. 

By analyzing the **Abstract Syntax Trees (ASTs)** of code generated in response to "Truth" (solvable) versus "Hallucination" (impossible) prompts, we aim to determine if high AST divergence (structural instability) is a reliable predictor of model error.

## 📂 Project Structure

* **`generate_questions.py`**: Generates the dataset (`research_data.json`) containing paired prompts (Truth vs. Hallucination) for the experiment.
* **`hf_models.py`**: The inference engine. It queries Hugging Face Inference Endpoints (e.g., Qwen 2.5, Llama 3.2) using the generated prompts and saves the raw code outputs to `final_cloud_results.json`.
* **`analyze_results.py`**: The analysis logic. It parses the ASTs of the generated code, calculates structural divergence metrics (Tree Edit Distance/Entropy), and generates visualization graphs.
* **`graphs/`**: Stores the output visualizations (e.g., `graph_Qwen-2.5-7B.png`).
