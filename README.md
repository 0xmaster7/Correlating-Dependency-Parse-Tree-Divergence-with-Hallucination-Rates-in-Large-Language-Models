# Correlating Abstract Syntax Tree (AST) Divergence with Hallucination Rates in LLMs

This research project investigates the structural stability of code generated by Large Language Models (LLMs). We specifically analyze whether the **Abstract Syntax Tree (AST)** of a response can serve as a proxy for detecting hallucinations.

Current findings for models like **Llama-3.2-3B** and **Qwen-2.5-7B** suggest they are "Perfect Liars"—maintaining high grammatical integrity (measured by Tree Depth) even when generating entirely hallucinated content.

## 📂 Data & JSON Structure

* **`research_data.json`**: The core dataset containing 2,000 prompts: 1,000 "Truth" questions (solvable tasks) and 1,000 "Hallucination Triggers" (impossible tasks). This part was generated by an LLM so we could optimize this further if needed but it serves as a baseline
* **`cloud_results_partial.json`**: A temporary checkpoint file used to save progress incrementally during API calls to prevent data loss from timeouts.
* **`final_cloud_results.json`**: The complete repository of raw code outputs from all tested models (Llama-3.2-3B and Qwen-2.5-7B) for both prompt categories. This serves as the primary input for analysis. For the final paper we will probably have to add more models but for now this is a decent baseline.

## 🛠️ Project Architecture

* **`generate_questions.py`**: Populates `research_data.json` with the experimental prompt set.
* **`hf_models.py`**: Queries the Hugging Face Inference API for each prompt and compiles the responses into the final results JSON.
* **`analyze_results.py`**: Parses the ASTs of the generated code to calculate structural divergence and generates visualization graphs (e.g., `graph_Qwen-2.5-7B.png`).
